# common operation

## common

| Main Function | Hot Key      | Remark      |
| ------------- | ------------ | ----------- |
| common        | Ctrl+P       | 快速打开最近打开的文件 |
| common        | Ctrl+Shift+N | 打开新的编辑器窗口   |
| common        | Ctrl+Shift+W | 关闭编辑器       |
| common        | ctrl + z     | 回退          |
